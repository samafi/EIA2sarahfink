Fragen und Antworten

1. Was beschreibt das Interface AssocStringString und wozu kann es verwendet werden?
Das Interface ist ein homogenes assiziatives Array, dem ein "Key" als Schlüssel vom Typ string hinzugefügt wird. Der zurückgegebene Wert ist ein string.

2. Was geschieht in Zeile 9 und was ist das Ergebnis?
Die Variable port vom Typ number wird erstellt. Durch process.env.port wird die Umgebung nach Informationen zum port durchsucht. Falls nichts gefunden wird, wird der port auf 1800 gesetzt. Der port dient als Kennnummer, um auf den localhost zuzugreifen.

3. Was geschieht in Zeile 25 von ServerTest.ts?
let query: AssocStringString = Url.parse(_request.url, true).query;
Es wird die Variable 'query' erstellt vom Typ des 'AssocStringString' Interfaces das einen Schlüssel als string erwartet.
Die 'Url.parse()' wandelt einen String vom URL um und gibt ein URL-Objekt zurück.
Wenn 'Url.parse()' true ist, dann wird aus query ein neues Objekt vom Interface 'AssocStringString'
Die '_request.url' (Anfrage an Server) kann dann das angefügte 'query' als Objekt (also a und b) ausgegeben.

4. Wie arbeitet die for-in-Schleife in Zeile 29?
Die Schleife sucht die Key-words (des AssocStringString) im der Variable query ab und gibt sie auf der Konsole aus. Teile des Arrays, welche nicht den Schlüssel key enthalten werden nicht beachtet.

5. Was bewirkt die Header-Information Access-Control-Allow-Origin?
Sie erlaubt dem Client den Inhalt dieser Seite im Header(Origin) darzustellen. Es ist ein Mechanismus, der dem Browser eine Kommunikation erlaubt. Es werden Freigabeeinstellungen bestimmt, wodurch man auf die Inhalte zugreifen kann.
